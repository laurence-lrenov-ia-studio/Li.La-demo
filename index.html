Départements actifs

Liste des territoires actifs sur Li·La. Pour ouvrir un nouveau département, ajoutez-le ici puis créez son URL Tally d'inscription événements ci-dessous.

Formulaires d'inscription Tally

1 URL Tally par département pour les événements (Apér'Immo / Pause Café), 1 URL Tally globale pour les masterclass (toutes antennes). Les boutons "S'inscrire" transmettent les paramètres pour que Make sache quel event/membre relier.

Enregistré

Champs à transmettre dans Tally

Pour que Make sache à quel événement et quel membre relier l'inscription, configurez ces "Hidden fields" dans chacun de vos formulaires Tally. L'app transmet ces valeurs en query string.

Param URL Valeur transmise À récupérer dans Make
event_id ou masterclass_idID de l'event (ex: E001)Pour identifier l'event
titreTitre de l'eventPour le mail confirmation
dateDate ISO (YYYY-MM-DD)Pour les rappels
departementCode département (ex: 94)Pour le routage admin
licencie_idID du membre (ex: AVA-00013)Pour relier au profil
licencie_nomPrénom + NomPour pré-remplir Tally

Administrateurs

Deux niveaux d'administration :

  • Super-admin — Linda et Laurence : accès total (réseau entier, masterclass, liens, paramètres).
  • LM (Licencié de Marque) — un membre délégué à la gestion d'un département : modifie les fiches et les événements de SON département uniquement. Ne touche pas aux masterclass, liens utiles ou paramètres.

Pour nommer un nouveau LM : convention LM + code département (ex : LM33, LM75). La gestion complète sera disponible une fois l'app reliée au Sheet (V1 fonctionnelle).
